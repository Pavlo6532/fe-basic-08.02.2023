Lesson 3 (Git Hub)

Форматування коду. Emmet. Валідація HTML

  • git status - дозволяє подивитися які зміни у репозиторії;
  • git add . - дозволяє зберегти зміни у робочому каталозі;
  • git commit -m - команда для створення коміту (прапор -m дозволяє не відкривати термінал за замовчуванням);
  • git push - команда для відправки з локального репозиторія до віддаленого;
  • git remote - дозволяє подивитися список підключених репозиторіїв;
  • git pull - команда для завантаження віддаленого репозиторію в локальний;
  • git checkout - команда для переміщення між гілками;
  • git branch - команда для роботи з гілкамі (створення, видалення, перейменовування).
